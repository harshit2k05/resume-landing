## 📄 About
ResumeFlow is a clean, modern, and professional landing page for an AI-powered resume builder. The design is fully responsive, ensuring a seamless user experience across desktop, tablet, and mobile devices.

## ⚡ Features
* **Semantic HTML**: Built using modern, accessible elements (`<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<footer>`).
* **Fully Responsive**: Optimized for various screen sizes using CSS media queries.
* **Modern UI**: Styled with Inter font for a professional look and feel.
* **Dynamic Content**: JavaScript integration for automated footer year updates.

## 🛠️ Technologies Used
* HTML5
* CSS3
* JavaScript
